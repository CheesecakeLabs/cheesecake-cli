# CheesecakeLabs Cli

## How to use this cli

`ckl <command> <args?>`

> parameters with < ?> are optionals

## Commands

`ckl new <customer?> <project?>` Create new project
`ckl add <stack?> <project?>` Add a new starter on a project
`ckl utils install <util-name?>` Install utils
`ckl help` Show help screen
